# Deploy PhysioByRutvi To Hostinger

1. Open Hostinger hPanel.
2. Go to Websites -> physiobyrutvi.in -> File Manager.
3. Open `public_html`.
4. Back up the current website files.
5. Delete old website files from `public_html`.
6. Upload `PhysioByRutvi_Hostinger_Deploy.zip`.
7. Extract the ZIP inside `public_html`.
8. Confirm `index.html` is directly inside `public_html`, not inside a nested folder.
9. Confirm `.htaccess`, `assets/`, and the `.dc.html` files are directly inside `public_html`.
10. Visit `https://physiobyrutvi.in`.
11. If the old page still appears, clear browser cache and Hostinger cache.

Do not upload `.git`, `screens`, `uploads`, `node_modules`, or source-only files to `public_html`.
